# AI Voice Assistant

## Overview

AI Voice Assistant is a Python-based desktop assistant that can execute voice and text commands, open websites, perform searches, provide system information, and respond using text-to-speech.

## Features

* Voice Command Support
* Text-to-Speech Responses
* Google Search
* Wikipedia Search
* Open Websites
* Open Applications
* Tell Time and Date
* System Information
* Battery Status
* Joke Generator
* Command History
* Modern GUI using CustomTkinter

## Technologies Used

* Python
* CustomTkinter
* SpeechRecognition
* pyttsx3
* Wikipedia
* psutil
* pyjokes

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
python assistant.py
```
