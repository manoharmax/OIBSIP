# AtmosIQ - Advanced Weather Intelligence App

A modern Python-based weather intelligence application developed as part of the Oasis Infobyte Python Programming Internship.

## Features

* Real-time weather information
* Temperature and humidity tracking
* Wind speed monitoring
* Atmospheric pressure analysis
* Modern graphical user interface
* Environment variable based API security
* OpenWeather API integration

## Technologies Used

* Python 3.14
* CustomTkinter
* Requests
* OpenWeather API
* Python Dotenv
